# 💰 Economy Insight

Um sistema simples e eficiente de gerenciamento de finanças pessoais desenvolvido em **Java**. O projeto tem como objetivo consolidar despesas, calcular descontos e apresentar o saldo líquido final do usuário.

## 🚀 Funcionalidades Atuais
- Registro de despesas fixas e variáveis.
- Cálculo de deduções sobre o salário bruto.
- Apresentação do balanço final (Receitas x Despesas).
- Formatação de saída com precisão de moedas (R$).

## 📂 Estrutura do Projeto
O código foi modularizado para facilitar a leitura e manutenção:
* **Bloco01:** Classe responsável por armazenar os dados de **Despesas** e **Descontos** (variáveis estáticas).
* **Bloco02:** Classe responsável por processar o **Salário Bruto** e calcular o **Salário Líquido** com base nas informações do Bloco 01.
* **Bloco03:** É o módulo principal (`main`). Responsável por orquestrar a aplicação, calcular o saldo restante e exibir o painel final no console.
